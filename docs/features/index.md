# Features

Filetree provides a focused set of features for self-hosted file management. No database, no cloud lock-in — just a single binary and a config file.

## Overview

| Feature | Description |
|---------|-------------|
| [Browse & manage](browse-manage.md) | Folder tree, list/grid views, create/rename/move/copy/delete, trash, drag-and-drop |
| [Rich previews](rich-previews.md) | In-browser preview for images, video, audio, PDF, Markdown, JSON, CSV, HTML, text |
| [Secure auth](secure-auth.md) | Google/GitHub OAuth and local users, JWT sessions |
| [Signed URLs](signed-urls.md) | Time-limited HMAC-signed links for preview and download |
| [Simple admin](simple-admin.md) | YAML/JSON config, admin UI, no database |
